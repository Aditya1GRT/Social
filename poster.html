Introduction
The Social Scoop is a full-featured social media platform enabling users to connect, share, interact and build communities in real time.
Users can create posts with images & videos, react with likes, comment, follow/unfollow, and chat in real time.
Features voice & video calling via WebRTC, emoji messaging, and media sharing in conversations.
Provides a seamless, responsive experience across all devices with adaptive dark & light themes.
A privacy-first design: follow requests, private follower lists, and secure JWT authentication.
🎯
Objectives
01
Develop a full-featured, production-ready social media application.
02
Enable real-time interaction via Socket.IO messaging and notifications.
03
Provide secure, scalable infrastructure with MongoDB Atlas and Cloudinary.
04
Implement peer-to-peer voice & video calls using WebRTC technology.
05
Ensure privacy with follow-request approval and protected data.
06
Deliver a mobile-first responsive UI with modern glassmorphism design.

Key Features
👤
Authentication & Profiles
JWT-secured signup/login with customisable profiles, bio, and profile pictures.
📸
Posts with Media
Create, like, and comment on posts with images and videos stored on Cloudinary.
💬
Real-time Messaging
Socket.IO chat with emoji picker, photo/video sharing, and typing indicators.
📞
Voice & Video Calls
Peer-to-peer WebRTC calls with mute/camera controls and incoming call UI.
🔔
Smart Notifications
Real-time notifications for likes, comments, and follows with unread badge.
🌙
Dark & Light Themes
Persistent per-user theme preference synced across sessions and devices.
🔒
Privacy Controls
Follow request approval system; followers/following lists visible only to the owner.
✅
Conclusion
The Social Scoop demonstrates a complete, production-grade social platform built with a modern MERN stack and real-time capabilities.
It successfully integrates authentication, media handling, live messaging, and WebRTC calling — delivering a seamless user experience.
The glassmorphism UI, responsive design, and dark/light theming provide an engaging, accessible interface across all devices.
With continuous improvement, The Social Scoop can scale into a powerful global connectivity and community-building platform.
